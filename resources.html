Courses:

  • CS50x - Harvard's introduction to computer science, the course I started with; explains the basic CS concepts in easy to understand terms; 2.5h long video lectures, some short videos, good notes and a very challenging homework section where you get to put things into practice, mostly in C and Python
  • The Odin Project - made by developers, for developers; gives you everything you need to be able to get a job in the field; mostly reading, some external links to videos; many external resources linked at the end of each lesson
  • 100devs - An online video-based bootcamp; slow-paced videos with a lot of repetition meant to help you remember things; very comprehensive explanations of everything
  • Angela Yu's Complete WebDev Bootcamp - The only course I've paid for so far; haven't used it much, so I'll write more about it when I do
