Computer Science

HTML

HyperText Markup Language - the one the web browser reads documents in to display their content.

Its purpose is to give that content structure.

Each piece (or element) of content (text, images, etc) is marked with <tags>. Those tags mark the different elements (paragraphs, headings, links, etc.).

HTML documents follow a certain structure in order to be read by the browser correctly.

Each HTML doc starts by declaring the version of HTML which at the moment is <!DOCTYPE html> for HTML 5.

Right underneath, we signify the beginning of the actual document with the <html> element. All the rest of the page is contained within that element.

It's divided into two main elements - the <head> and the <body>.

In the <head> we include all information for the browser itself; it remains invisible to the user (with its <title> being the one notable exception).

The <body> on the other hand contains everything that the user will see.

CSS

Cascading Style Sheets - the one responsible for the presentation of a document's content.

Its purpose is to give that content style.
